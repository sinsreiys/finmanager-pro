# FinManager PRO

[![made-with-javascript](file:///C:/Users/locas/OneDrive/Escritorio/financial-managment-prt/backend/index.html)](https://www.javascript.com)

**FinManager PRO** es una aplicación web integral para la gestión de finanzas personales y empresariales. Diseñada para funcionar completamente en el navegador, almacena todos los datos de forma local (sin necesidad de backend), ofreciendo un dashboard potente, múltiples módulos y una experiencia totalmente personalizable.

🔗 **Demo en vivo:** [http://127.0.0.1:5001/) *(activa después de desplegar en GitHub Pages)*

## ✨ Características principales

- 📊 **Dashboard interactivo** con estadísticas de ingresos/gastos, gráficos circulares y patrimonio neto.
- 💰 **Gestión de transacciones** (ingresos/gastos) con filtros, categorías personalizables y etiquetas de presupuesto.
- 💳 **Tarjetas de crédito/débito** – control de saldo, límites, pagos e historial.
- 📄 **Facturas** – subida de archivos (PDF/imagen) y creación de facturas manuales con cliente, número, total, etc.
- 📅 **Agenda de eventos** – calendario interactivo para recordar fechas importantes.
- 🎯 **Presupuestos mensuales** – por categoría, estilo de vida (familia, mascotas, ocio, empresa) o personalizados.
- 🎯 **Metas de ahorro** – seguimiento de objetivos, aportes y progreso.
- 💸 **Préstamos** – registro de préstamos (principal, interés, plazo), abonos e historial.
- 🏷️ **Categorías personalizables** – añade/elimina categorías de ingresos y gastos.
- 👥 **Multiusuario y empresas** – permite múltiples empresas por usuario, roles (admin/miembro) y gestión de miembros.
- 🏭 **Módulos empresariales** – crea módulos a medida con contenido editable (Nóminas, Proveedores, etc.).
- 💾 **Backup y restauración** – descarga/restaura todos los datos en formato JSON.
- 📎 **Exportación a CSV** – transacciones y facturas.
- 🎨 **Temas visuales** – 5 temas: Claro (crema), Oscuro, Azul, Cian y Morado. Selector desplegable en la barra superior.
- 📱 **Diseño responsive** – adaptable a móviles y tablets.
- 🔐 **Autenticación local** – registro/login con persistencia de usuarios y empresas.

## 🛠️ Tecnologías utilizadas

- HTML5 / CSS3 (variables CSS, Flexbox, Grid)
- JavaScript (ES6+)
- [Chart.js](https://www.chart.js/) – gráficos interactivos
- LocalStorage – persistencia de datos
- FontAwesome / íconos emoji (sistema operativo)
